📊 Netflix Data Analysis

This project explores the Netflix Movies and TV Shows dataset to uncover trends in content distribution, genres, ratings, and release patterns.
Using Python-based data analysis techniques, the project cleans, analyzes, and visualizes the dataset to generate insights about Netflix's content library.

The analysis focuses on understanding how Netflix content is distributed across years, countries, and categories.

🎯 Project Objectives

##The main goals of this project are:

Analyze the distribution of Movies vs TV Shows

Identify top genres and content categories

Study content trends across different years

Explore country-wise content production

Understand ratings and duration patterns

🗂 Dataset

The dataset used in this project contains information about Netflix titles including:

Title

Type (Movie / TV Show)

Director

Cast

Country

Date Added

Release Year

Rating

Duration

Genre

📈 Sample Insights

Some interesting observations from the dataset:

Movies dominate the Netflix library compared to TV Shows.

Content growth peaked around 2017–2019.

The United States and India are among the largest content contributors.

Genres such as Drama and Documentaries appear frequently.
